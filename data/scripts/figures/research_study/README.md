This folder contains the independent research study developed from this project.

The study investigates how different climate-related stressors reshape microbial community structure across coral reef and soil ecosystems using publicly available microbiome datasets.

## Study Structure

The research study will follow a standard scientific structure:

1. Abstract  
2. Introduction  
3. Methods  
4. Results  
5. Discussion  
6. Conclusion  
7. References  

## Note

This work represents an independent exploratory research study conducted using publicly available datasets.  
It has not undergone formal peer review and is intended as an academic research project.
