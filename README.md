Coral Microbiome Analysis
This section focuses on analyzing microbial community structure in coral samples using correlation-based network analysis.

Approach
- Data preprocessing and cleaning  
- Removal of low-variance features  
- Correlation analysis to identify microbial relationships  
- Network construction to visualize interactions  
- Identification of hub taxa using degree centrality  

Interpretation
The resulting network reveals modular clusters of co-occurring microbes and identifies key taxa that may play important ecological roles within the coral microbiome.
